# Macallan 18A

## 项目介绍

    背景：现有的电源管理测试中需测试不同充电类型的充电电量、电流、充电时长等，测试耗时长，不同充电类型测试对比操作性差。
    目的：通过软硬件的开发，实现充电兼容和电量跳变的自动化测试，能够完成自动充放电，充电器的自动切换，通过对图像和Log的分析检查电流，电压等数据判断测试结果
